
# Vimdeck
## The "Action Bar" for Neovim.

Currently a work in progress.

## Installation

### Lazy
```
   {
    'AWoelfle/vimdeck.nvim',
    requires = { 'nvim-lua/plenary.nvim'}
   }
```

### Vim Config
```
require("vimdeck").setup({})
```
